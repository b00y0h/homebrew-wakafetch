# Homebrew Tap for wakafetch

Homebrew tap for [wakafetch](https://github.com/b00y0h/wakafetch) - a modern system information tool.

## Installation

```bash
brew tap b00y0h/wakafetch
brew install wakafetch
```

## About

This tap provides the wakafetch formula for Homebrew installation on macOS and Linux.
